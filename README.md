# My Personal Website

This is my personal website hosted on GitHub Pages. It serves as a hub for various experiments and projects.

## Structure

- `index.html`: Main page with links to experiments
- `styles.css`: Shared styles
- `travel/`: Travel blog experiment
- `cooking/`: Cooking website experiment

## Adding New Experiments

To add a new experiment:
1. Create a new directory (e.g., `newexperiment/`)
2. Add an `index.html` file in that directory
3. Update the main `index.html` to include a link to the new experiment
